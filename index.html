Enapps Ltd , London OpenERP partner have completely rewritten the core module account_voucher with many great enhancements in usability, functionality, technical design and features. This module is a full replacement of the standard account_voucher supplied with OpenERP. Remove the old, install the new and enjoy!

  • Multi-currency payments and write-off
  • 10 times faster
  • Simple and powerful user interface
  • Invoice and credits on the same line

Account voucher module by Enapps is a full rewrite of the original account_voucher with clean code and improved functionality. This module is to be used to record single payment transactions to suppliers or from customers. This functionality will be very familiar to Quickbooks and Oracle users. The module fully supports multicurrency.

Versions

Current stable 6.1 version is tested and works ideally with Enapps Web server and Enapps OpenERP core server, V7.0 support is in process.
